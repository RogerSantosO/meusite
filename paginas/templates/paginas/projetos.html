Análise de ações

Esta é uma aplicação web feita com a biblioteca streamlit. Além de ser um biblioteca de fácil compreensão e com opções úteis ela permite que você faça o deploy do projeto gratuitamente.

Requisitar dados online

O acesso a certos dados online pode ser muito extenso e demorado, mas com a ferramenta certa você pode obtê-los em segundos e com uma análise pré-definida. Neste projeto eu precisava de alguns preços de uma loja de roupas e requisitei os dados para serem retornados em um arquivo txt

Análise com pandas

Pandas é uma biblioteca que auxilia na análise de dados em tabela e junto com outras bibliotecas também permite fazer a visualização dos dados de forma organizada. Neste projeto analisei dados de jogos de diversas plataformas.
